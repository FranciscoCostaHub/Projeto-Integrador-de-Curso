# Projeto-Integrador-de-Curso
Final Bsc Eletrical and Computer Engineering Project, using TDoA mathematics method applied to microphone captated signals to estimate its direction

O projeto "GUNSHOT DETECTION SYSTEM" pretende implemetar uma solução de baixo custo, capaz de fornecer às forças policiais e militares uma maior "situational awareness", indicando posições de disparos inimigos. A implementação utiliza microcontroladores ESP32, Raspberry Pico 2W, 4 MEMS I2S e um ecrã Pimoroni Pico Display.
